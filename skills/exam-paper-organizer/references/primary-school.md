# 小学组卷指导

## 小学语文

### 考试结构（100分/60-90分钟）

| 题型 | 分值 | 考查内容 |
|-----|------|---------|
| 基础知识 | 30-35分 | 拼音、汉字、词语、句子 |
| 积累运用 | 20-25分 | 古诗文默写、课文内容 |
| 阅读理解 | 25-30分 | 课内/课外阅读 |
| 习作表达 | 25-30分 | 命题/半命题作文 |

### 各年级重点

**低年级（1-2年级）：**
- 拼音：声母、韵母、整体认读音节
- 识字：常用字800个左右
- 写字：笔画、笔顺、间架结构
- 阅读：简单记叙文理解
- 写话：看图写话、简单句子

**中年级（3-4年级）：**
- 字词：多音字、形近字、近反义词
- 句子：扩句、缩句、修改病句
- 阅读：概括段落大意、理解中心思想
- 习作：记叙文（记事、写人、写景）

**高年级（5-6年级）：**
- 词语：成语、俗语、关联词运用
- 句子：修辞手法、句式转换
- 阅读：说明文、议论文初步，赏析评价
- 习作：综合表达，思想性、条理性

### 组卷要点

1. **基础题占比60%**：拼音、汉字、词语是重点
2. **阅读题选材**：课内为主，课外为辅，篇幅适中
3. **作文题目**：贴近生活，有写作空间
4. **难度梯度**：由易到难，给学生信心

## 小学数学

### 考试结构（100分/60-90分钟）

| 题型 | 分值 | 特点 |
|-----|------|------|
| 填空题 | 20-25分 | 基础概念、简单计算 |
| 选择题 | 10-15分 | 概念辨析、估算判断 |
| 判断题 | 5-10分 | 易错概念辨析 |
| 计算题 | 25-30分 | 口算、笔算、简算、解方程 |
| 操作题 | 5-10分 | 作图、测量 |
| 解决问题 | 25-30分 | 应用题、综合实践 |

### 各年级核心知识点

**一年级：**
- 数：20以内/100以内数的认识与运算
- 图形：认识基本图形
- 应用：简单加减法应用题

**二年级：**
- 数：表内乘除法、万以内数的认识
- 测量：长度单位、质量单位
- 应用：乘除法应用题

**三年级：**
- 数：分数初步、小数初步、四则运算
- 图形：长方形和正方形周长面积
- 应用：两步计算应用题

**四年级：**
- 数：大数认识、运算律、小数运算
- 图形：角的度量、平行四边形和梯形
- 统计：条形统计图

**五年级：**
- 数：因数倍数、分数意义与运算
- 图形：长方体和正方体（表面积、体积）
- 应用：方程解应用题

**六年级：**
- 数：分数/百分数/比和比例
- 图形：圆、圆柱圆锥
- 统计：扇形统计图
- 综合：小升初衔接内容

### 组卷要点

1. **计算占40%**：基础计算能力是核心
2. **应用题生活化**：贴近学生生活经验
3. **操作题不可少**：培养空间观念
4. **难度控制**：基础70%、中等20%、提高10%

## 小学英语

### 考试结构（100分/60分钟）

| 题型 | 分值 | 考查能力 |
|-----|------|---------|
| 听力 | 30分 | 听辨、理解、反应 |
| 词汇语法 | 25分 | 单词拼写、句型转换 |
| 阅读理解 | 20分 | 篇章理解 |
| 书面表达 | 15分 | 写作 |
| 口语（单独） | 10分 | 朗读、对话 |

### 各年级要求

**三年级（起始年级）：**
- 字母：大小写、发音
- 词汇：约100个核心词
- 句型：简单问候、介绍
- 听说为主，读写跟进

**四年级：**
- 词汇：累计250-300词
- 语法：一般现在时、现在进行时
- 阅读：简单对话、短文
- 写作：仿写句子、短文

**五年级：**
- 词汇：累计500词左右
- 语法：一般过去时、比较级
- 阅读：篇章理解
- 写作：命题作文5-8句

**六年级：**
- 词汇：累计700词左右
- 语法：综合时态运用
- 阅读：较长篇章
- 写作：命题作文8-10句

### 组卷要点

1. **听力先行**：听力测试放第一部分
2. **词汇语境化**：避免孤立考查单词
3. **阅读篇幅适中**：100-150词为宜
4. **写作给支架**：提供关键词、句型提示

## 小学科学（3-6年级）

### 考试结构（100分/60分钟）

| 题型 | 分值 |
|-----|------|
| 填空题 | 30分 |
| 选择题 | 20分 |
| 判断题 | 10分 |
| 实验探究 | 25分 |
| 综合应用 | 15分 |

### 内容领域

- 生命科学：植物、动物、人体
- 物质科学：物理、化学初步
- 地球与宇宙科学：天气、地球、宇宙
- 技术与工程：设计与制作

### 组卷要点

1. **实验探究是重点**：体现科学学科特点
2. **联系生活**：科学就在身边
3. **图文并茂**：配有插图、图表
4. **开放性试题**：鼓励创新思维
