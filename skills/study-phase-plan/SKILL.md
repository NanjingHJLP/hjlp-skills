# 学习阶段计划

## 概述

本 Skill 用于为用户制定结构化的分阶段学习计划，采用**基础阶段→强化阶段→冲刺阶段**的三段式模型，适用于小学、初中、高中、大学各学段及各学科课程。

## 三阶段模型

### 基础阶段
**目标**：系统掌握知识点，建立完整知识框架
- **核心任务**：通读教材、理解概念、整理笔记、建立知识框架
- **时间占比**：总学习周期的 40-50%
- **典型活动**：
  - 逐章学习教材内容
  - 整理知识框架图/思维导图
  - 完成基础练习题
  - 标记疑难知识点

### 强化阶段
**目标**：深化理解，突破重难点，提升解题能力
- **核心任务**：专题突破、重难点攻克、题型归纳、错题整理
- **时间占比**：总学习周期的 30-35%
- **典型活动**：
  - 重点章节深度学习
  - 专题训练（如函数专题、力学专题）
  - 历年真题分类练习
  - 错题本整理与回顾
  - 解题方法归纳

### 冲刺阶段
**目标**：查漏补缺，模拟实战，调整到最佳状态
- **核心任务**：全真模拟、查漏补缺、状态调整、应试技巧
- **时间占比**：总学习周期的 15-25%
- **典型活动**：
  - 全真模拟考试
  - 重点背诵/记忆内容
  - 查漏补缺（针对薄弱点）
  - 回顾错题本
  - 调整作息与心态

## 学段特点

### 小学阶段
- **学习特点**：形象思维为主，需多感官参与
- **阶段侧重**：基础阶段更重趣味性，强化阶段注重习惯养成
- **里程碑**：单元测试、期中/期末考试

### 初中阶段
- **学习特点**：学科分化明显，抽象思维开始发展
- **阶段侧重**：基础阶段注重概念理解，强化阶段培养解题思维
- **里程碑**：月考、期中/期末、中考（初三）

### 高中阶段
- **学习特点**：知识密度大，抽象思维要求高，应试导向明确
- **阶段侧重**：三阶段均需高强度，强化阶段专题突破尤为关键
- **里程碑**：月考、联考、期中/期末、高考（高三）

### 大学阶段
- **学习特点**：自主学习为主，知识深度与广度兼具
- **阶段侧重**：基础阶段建立理论体系，强化阶段结合实践应用
- **里程碑**：阶段性测验、期中/期末、考研/考证（备考期）

## 学科特点

### 理科类（数学、物理、化学、生物）
- **基础阶段**：公式定理推导理解、概念辨析、基础计算
- **强化阶段**：题型分类训练、综合题突破、实验题型（理化生）
- **冲刺阶段**：易错点回顾、计算准确性训练、解题规范

### 文科类（语文、英语、历史、政治、地理）
- **基础阶段**：词汇/古诗文积累、知识点梳理、阅读基础
- **强化阶段**：阅读理解专项、写作训练、材料分析能力
- **冲刺阶段**：重点背诵、作文素材、答题模板

### 工科类（计算机、工程类课程）
- **基础阶段**：语法/原理学习、基础操作、简单项目
- **强化阶段**：算法/设计模式、综合项目、调试能力
- **冲刺阶段**：项目优化、常见问题排查、代码规范

## 执行流程

当用户提出学习阶段计划需求时，按以下步骤执行：

1. **确认信息**：
   - 学段（小学/初中/高中/大学）
   - 学科/课程名称
   - 当前年级/学期
   - 目标（期中/期末/升学/考证等）
   - 可用学习时间（天数/周数）
   - 当前水平（基础薄弱/中等/较好）

2. **制定三阶段计划**：
   - 根据总时间分配三阶段时长
   - 为每阶段设定具体目标和任务
   - 列出每阶段的核心学习内容
   - 设置阶段检查点（里程碑）

3. **输出格式**：
   - 阶段划分与时间安排
   - 每阶段学习目标
   - 具体学习任务清单
   - 里程碑检查点
   - 学习建议与注意事项

## 输出模板

```
## 【学段+学科+目标】学习计划

### 基本信息
- 学段：高中二年级
- 学科：物理
- 目标：高二上学期期末复习
- 周期：8周
- 当前水平：中等

### 阶段一：基础阶段（第1-3周）
**目标**：系统梳理上半学期知识点，建立完整知识框架

**核心任务**：
1. 逐章复习教材（电场、电路、磁场）
2. 整理各章知识框架图
3. 完成基础概念辨析
4. 基础公式推导与记忆

**每日安排建议**：
- 工作日：2小时（1小时复习+1小时练习）
- 周末：4小时（系统梳理+错题整理）

**里程碑检查点（第3周末）**：
- [ ] 完成全部章节知识点梳理
- [ ] 基础练习题正确率≥70%
- [ ] 能独立画出各章知识框架

---

### 阶段二：强化阶段（第4-6周）
**目标**：专题突破，攻克重难点，提升综合解题能力

**核心任务**：
1. 专题训练（电场综合、带电粒子运动、电磁感应）
2. 历年真题分类练习
3. 错题归类分析
4. 压轴题解题方法归纳

**每日安排建议**：
- 工作日：2.5小时（1.5小时专题+1小时真题）
- 周末：5小时（专题突破+模拟测试）

**里程碑检查点（第6周末）**：
- [ ] 专题练习正确率≥75%
- [ ] 完成3套综合模拟卷
- [ ] 错题本整理≥50道典型题

---

### 阶段三：冲刺阶段（第7-8周）
**目标**：查漏补缺，全真模拟，调整到最佳状态

**核心任务**：
1. 全真模拟考试（严格限时）
2. 回顾错题本与薄弱点
3. 重点公式与结论记忆
4. 考试技巧与心态调整

**每日安排建议**：
- 工作日：2小时（1套限时练习+1小时回顾）
- 周末：全真模拟+试卷分析

**里程碑检查点（第8周末/考前）**：
- [ ] 模拟考试成绩达到目标分数
- [ ] 错题本全部回顾一遍
- [ ] 核心知识点100%掌握
- [ ] 作息调整至考试节奏

---

### 整体建议
1. **优先级**：电磁学综合题 > 基础概念 > 计算准确性
2. **资料推荐**：教材+学校练习册+5年真题
3. **注意事项**：
   - 每周日进行阶段复盘
   - 保持良好作息，避免熬夜
   - 遇到瓶颈及时寻求帮助
```

## 常见问题

**Q: 只有2周复习时间怎么安排？**
A: 压缩基础阶段至5天（只抓核心），强化阶段7天（专题突破），冲刺阶段2天（全真模拟+回顾）。

**Q: 基础薄弱如何调整？**
A: 延长基础阶段占比至60%，强化阶段重点抓中档题，冲刺阶段降低难度预期，确保基础分拿满。

**Q: 多科同时复习如何协调？**
A: 采用"主科深、副科浅"策略，主科完整三阶段，副科压缩为两阶段（基础+冲刺），或采用轮换复习法。

**Q: 计划执行不下去怎么办？**
A: 将里程碑检查点设为不可逾越的节点，未完成则暂停前进，分析原因调整计划，宁可进度慢也要保证质量。
