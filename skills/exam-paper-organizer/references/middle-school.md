# 中学组卷指导

## 初中语文

### 考试结构（120分/120分钟）

| 题型 | 分值 | 内容 |
|-----|------|------|
| 积累与运用 | 25-30分 | 字音字形、词语运用、病句、古诗文默写 |
| 古诗文阅读 | 15-20分 | 课内/课外文言文、古诗词鉴赏 |
| 现代文阅读 | 35-40分 | 记叙文、说明文/议论文 |
| 名著阅读 | 5-10分 | 必读名著考查 |
| 写作 | 50分 | 命题/半命题/材料作文 |

### 各年级特点

**七年级：**
- 文言文：课内为主，寓言、传记类
- 现代文：记叙文阅读为重点
- 写作：记叙文写作，注重真情实感
- 名著：《朝花夕拾》《西游记》

**八年级：**
- 文言文：增加议论性文章
- 现代文：说明文加入，记叙文加深
- 写作：说明文写作训练
- 名著：《红星照耀中国》《昆虫记》

**九年级：**
- 文言文：综合各类文体，难度提升
- 现代文：议论文加入，综合训练
- 写作：议论文初步，中考冲刺
- 名著：《水浒传》《简爱》等

### 组卷要点

1. **中考导向**：研究本地中考命题风格
2. **阅读选材**：文质兼美，体现时代性
3. **作文命题**：避免套作，有区分度
4. **名著考查**：避免偏题怪题，关注核心情节

## 初中数学

### 考试结构（120分/120分钟）

| 题型 | 分值 | 题量 |
|-----|------|------|
| 选择题 | 30分 | 10题×3分 |
| 填空题 | 18分 | 6题×3分 |
| 解答题 | 72分 | 8-9题 |

### 各年级核心内容

**七年级：**
- 有理数、整式、一元一次方程
- 几何图形初步、相交线与平行线
- 实数、平面直角坐标系
- 二元一次方程组、不等式
- 数据的收集与整理

**八年级：**
- 三角形、全等三角形、轴对称
- 整式乘法与因式分解
- 分式、二次根式
- 一次函数、反比例函数
- 平行四边形、特殊平行四边形

**九年级：**
- 一元二次方程
- 二次函数（重点+难点）
- 旋转、圆
- 概率初步
- 相似、锐角三角函数
- 投影与视图

### 中考重点模块

1. **数与代数（约45%）**：方程、函数是核心
2. **图形与几何（约40%）**：三角形、圆、函数与几何综合
3. **统计与概率（约15%）**：基础送分题

### 压轴题设计

**常见压轴类型：**
- 二次函数与几何综合
- 动态几何问题
- 新定义问题
- 几何探究题

### 组卷要点

1. **基础题保分**：选择前8题、填空前4题、解答题前4题要基础
2. **中档题区分**：后2道选择、后2道填空、中间2道解答
3. **压轴题分层**：设置2-3小问，由易到难
4. **计算量适中**：避免过繁计算掩盖思维考查

## 高中语文

### 考试结构（150分/150分钟）

| 题型 | 分值 | 说明 |
|-----|------|------|
| 现代文阅读Ⅰ | 19分 | 信息类文本（论述/实用） |
| 现代文阅读Ⅱ | 16分 | 文学类文本（小说/散文） |
| 文言文阅读 | 20分 | 选择+翻译+概括 |
| 古代诗歌阅读 | 9分 | 选择+简答 |
| 名句默写 | 6分 | 6句 |
| 语言文字运用 | 20分 | 成语、病句、补写、修辞等 |
| 写作 | 60分 | 材料作文 |

### 新高考变化

1. **阅读量大**：整本书阅读、群文阅读
2. **题型灵活**：情境化命题增加
3. **作文思辨**：强调逻辑思维

### 组卷要点

1. **文本质量**：选文经典，有文化内涵
2. **题目梯度**：由理解到鉴赏到评价
3. **作文材料**：有思辨空间，避免一边倒
4. **关注时事**：语言运用题结合热点

## 高中数学

### 考试结构（150分/120分钟）
- **新高考（不分文理）：** 单选8×5+多选4×5+填空4×5+解答6题（77分）
- **老高考（文理分科）：** 选择12×5+填空4×5+解答6题（70分）

### 各年级核心内容

**高一（必修）：**
- 集合与逻辑、不等式
- 函数概念与性质、指数对数函数
- 三角函数
- 平面向量
- 复数

**高二（选择性必修）：**
- 数列
- 立体几何（向量法）
- 解析几何（直线与圆、圆锥曲线）
- 计数原理、概率统计
- 导数及其应用

**高三（复习）：**
- 一轮：系统复习全部内容
- 二轮：专题突破
- 三轮：综合模拟

### 高考重点

1. **函数与导数（约27分）**：压轴题常客
2. **解析几何（约22分）**：圆锥曲线综合
3. **数列（约10分）**：与函数、不等式综合
4. **三角（约17分）**：基础题为主
5. **立几（约22分）**：证明与计算
6. **概率统计（约17分）**：应用题趋势

### 压轴题设计

**新高考最后两题（通常）：**
- 倒数第二题：解析几何综合
- 最后一题：函数与导数综合

**设计原则：**
- 第(1)问基础，送分
- 第(2)问中等，有区分度
- 第(3)问（如有）拔高，选拔人才

### 组卷要点

1. **知识点覆盖**：大题考点尽量不重复
2. **难度梯度**：选择/填空由易到难排列
3. **解答题配置**：三角/数列、立几、概率、解几、导数
4. **计算合理**：避免纯繁琐计算
5. **新题型**：情境题、开放题适量

## 初中英语

### 考试结构（120分/120分钟）

| 题型 | 分值 |
|-----|------|
| 听力 | 25分 |
| 阅读理解 | 40分（4篇） |
| 语言知识运用 | 25分（完形+语法填空） |
| 写作 | 20分（应用文+续写/概要） |
| 口语（单独） | 10分 |

### 中考词汇要求
- 课标词汇：1600词左右
- 短语：300个左右
- 重点语法：时态、语态、从句

### 组卷要点

1. **听力语速**：120词/分钟左右
2. **阅读题材**：多样（故事、科普、文化）
3. **完形主题**：励志、成长、文化
4. **写作情境**：真实交际情境

## 高中英语

### 考试结构（150分/120分钟）

| 题型 | 分值 |
|-----|------|
| 听力 | 30分 |
| 阅读理解 | 50分（5篇，含七选五） |
| 语言知识运用 | 30分（完形+语法填空） |
| 写作 | 40分（应用文15+读后续写25） |

### 高考词汇要求
- 课标词汇：3500词
- 阅读词汇：需达到4500-5000认知量

### 组卷要点

1. **阅读篇幅**：A篇200词，CD篇400词左右
2. **CD篇难度**：体现选拔性，长难句多
3. **七选五**：考查语篇衔接
4. **读后续写**：情节合理，语言丰富
5. **文化意识**：中外文化对比

## 其他学科

### 初中物理（100分/90分钟）

**内容分布：**
- 声光热：15%
- 力：40%
- 电：40%
- 其他：5%

**重点实验：**
- 测密度、探究杠杆平衡、测电阻/电功率

### 初中化学（100分/60分钟）

**内容分布：**
- 基本概念：20%
- 身边的化学物质：35%
- 化学计算：15%
- 化学实验：20%
- 化学与社会：10%

### 高中物理（100分/90分钟）

**新高考模式：**
- 选择（单选+多选）：40-46分
- 实验：10-15分
- 计算/论述：44-50分

**压轴：** 力学综合、电磁综合

### 高中化学（100分/90分钟）

**内容：**
- 必修：基本概念、元素化合物、反应原理
- 选择性必修：结构、有机、反应原理加深

### 初中/高中政治、历史、地理

按课标要求，注重核心素养考查，情境化命题，关注时事热点。
