# 申论专项训练

## 一、申论概述

### 1.1 考试特点

- **考试时间**：180分钟
- **题量**：4-5道题
- **满分**：100分
- **考查能力**：阅读理解、综合分析、提出和解决问题、文字表达

### 1.2 试卷结构

**材料部分：**
- 给定资料：6000-8000字，5-10则材料
- 材料类型：新闻报道、专家观点、政策文件、案例描述、数据统计

**题目部分：**
- 概括题（1-2道）
- 综合分析题（1道）
- 对策题或贯彻执行题（1道）
- 大作文（1道，1000-1500字）

### 1.3 核心能力要求

**阅读理解能力：**
- 理解材料主旨
- 把握给定资料的核心内容
- 准确理解材料中词语、句子的含义

**综合分析能力：**
- 对材料中的问题、观点进行分析
- 作出合理的推断或评价
- 对多则材料进行归纳整合

**提出和解决问题能力：**
- 针对问题提出切实可行的措施
- 措施要具体、可操作
- 符合政府职能和实际

**文字表达能力：**
- 准确、规范地进行表述
- 语言流畅，逻辑清晰
- 用词恰当，符合公文语体

## 二、材料阅读方法

### 2.1 材料类型识别

**1. 政策理论型**
- 特点：领导人讲话、政策文件、理论阐述
- 作用：定调子，给出答题方向
- 示例："习近平总书记指出..."

**2. 新闻报道型**
- 特点：叙述事件、介绍情况
- 作用：提供案例、反映问题
- 示例："近日，记者在XX地调查发现..."

**3. 专家观点型**
- 特点：专家学者、业内人士的看法
- 作用：分析问题原因、提出对策建议
- 示例："XX专家指出，这一问题的根源在于..."

**4. 数据统计型**
- 特点：数字、图表、对比
- 作用：证明问题严重性、展示成效
- 示例："据统计，2023年XX达到XX，同比增长XX%"

**5. 群众声音型**
- 特点：普通市民、网友的反映
- 作用：反映问题、表达诉求
- 示例："市民李先生告诉记者..."

### 2.2 阅读技巧

**第一步：看题目（2分钟）**
- 了解有几道题、什么题型
- 确定每道题的作答范围
- 做到心中有数，带着问题读材料

**第二步：通读材料（15-20分钟）**
- 快速浏览，了解材料大意
- 用横线、波浪线标注关键句
- 在材料旁标注关键词（问题、原因、对策、影响）
- 不需要精读，抓住大意即可

**第三步：精读材料（根据题目需要）**
- 针对具体题目，精读相关段落
- 提取要点，整理答案

**标注符号建议：**
- 横线：问题
- 波浪线：对策
- 圆圈：关键词
- 方框：数据
- 问号：疑问点
- 感叹号：重要观点

### 2.3 段落关系分析

**总分关系：**
- 首段总述，后段分述
- 或末段总结，前段分述

**并列关系：**
- 几段讲同一层面的内容
- 可以合并概括

**递进关系：**
- 现象→原因→对策
- 问题→影响→解决

**转折关系：**
- 前面说好，后面说问题
- 注意"但是""然而"等转折词

**因果关系：**
- 原因和结果分别标注

## 三、题型详解与答题方法

### 3.1 归纳概括题

**题型识别：**
- "概括..."
- "总结..."
- "归纳..."
- "指出...存在的问题"

**答题思路：**
1. 审题：明确概括对象（问题、原因、影响、做法等）
2. 找点：回材料定位相关信息
3. 提炼：去除例子、数据，保留核心内容
4. 整合：合并同类项，分条作答
5. 作答：总分结构，条理清晰

**答题格式：**
```
总括句：XX问题主要表现在以下几个方面：

一是...；
二是...；
三是...；
四是...。
```

**示例：**
> 请概括材料中反映的乡村教师队伍建设存在的问题。
>
> 【参考答案】
> 乡村教师队伍建设存在以下问题：
> 
> 一是待遇偏低。乡村教师工资水平低于城市，难以吸引优秀人才。
> 
> 二是条件艰苦。乡村学校基础设施落后，生活条件差。
> 
> 三是编制不足。乡村教师编制紧张，存在"超编缺人"现象。
> 
> 四是发展受限。乡村教师培训机会少，晋升渠道窄。
> 
> 五是流失严重。优秀乡村教师向城市流动，人才留不住。

**注意事项：**
- 要点全面，宁多勿少
- 语言简练，去掉修饰词
- 分条作答，不要写一大段
- 控制字数，不超过规定字数10%

### 3.2 综合分析题

**题型分类：**

**1. 解释概念型**
- 问法："谈谈你对XX的理解"
- 答题结构：是什么→为什么→怎么办
- 是什么：解释概念内涵
- 为什么：分析原因、意义、影响
- 怎么办：提出对策

**2. 评价观点型**
- 问法："谈谈你对XX观点的看法"
- 答题结构：表态→分析→总结
- 表态：赞同/反对/辩证看待
- 分析：论证观点，可用原因、影响、例子
- 总结：重申观点或提对策

**3. 比较分析型**
- 问法："比较A和B的异同"
- 答题结构：分相同点和不同点分别作答

**4. 启示分析型**
- 问法："XX做法给我们什么启示"
- 答题结构：概括做法+提炼启示

**答题格式（解释概念型）：**
```
XX是指...（内涵解释）。

这一现象/做法：
从积极方面看...（意义、作用）；
从消极方面看...（问题、危害）。

因此，我们应当...（对策建议）。
```

**示例（评价观点）：**
> 材料中提到"科技是一把双刃剑"，请谈谈你的看法。
>
> 【参考答案】
> 我认为"科技是一把双刃剑"这一观点是正确的，应当辩证看待科技发展。
> 
> 一方面，科技发展带来巨大福祉。科技推动生产力发展，提高经济效益；改善人民生活，提供便利服务；促进社会进步，解决诸多难题。
> 
> 另一方面，科技发展也带来一些问题。如技术滥用侵犯隐私、网络诈骗频发、人工智能导致失业等社会问题。
> 
> 因此，我们要坚持科技创新，同时加强科技伦理建设，完善法律法规，让科技更好地服务于人。

**注意事项：**
- 观点要明确，不要模棱两可
- 分析要全面，多角度思考
- 逻辑要清晰，层层递进

### 3.3 提出对策题

**题型识别：**
- "提出对策/建议"
- "如何解决...问题"
- "给出意见建议"

**对策来源：**
1. 材料直接给出：领导人讲话、政策文件、专家建议
2. 问题反推对策：从问题推出解决措施
3. 经验借鉴：从成功案例中提取

**答题思路：**
1. 审题：明确针对什么问题提对策
2. 找点：从材料中找已有的对策建议
3. 补充：根据问题反推对策
4. 整理：按主体或层面分类
5. 作答：分条列出，具体可行

**常用对策角度：**

| 角度 | 常用表述 |
|------|---------|
| 思想认识 | 提高认识、转变观念、加强宣传 |
| 法律法规 | 完善法律、严格执法、加强监管 |
| 制度机制 | 建立机制、完善制度、考核评价 |
| 投入保障 | 加大投入、资金支持、人才保障 |
| 教育培训 | 加强培训、提高素质、引进人才 |
| 监督管理 | 加强监管、强化问责、信息公开 |
| 技术支撑 | 科技赋能、信息化、智能化 |
| 社会参与 | 公众参与、社会监督、舆论引导 |

**对策表述模板：**
```
一是完善法律法规。制定XX法，明确XX责任，为XX提供法律保障。

二是加强监督管理。建立XX机制，定期开展XX检查，对违规行为严肃查处。

三是加大资金投入。设立专项资金，支持XX建设，保障XX需要。

四是加强宣传教育。通过多种渠道宣传XX知识，提高公众XX意识。

五是建立长效机制。将XX纳入考核体系，定期评估，持续改进。
```

**注意事项：**
- 对策要具体，不要空洞
- 符合政府职能（政府不能做的事不要提）
- 一条对策一个要点，不要混着说
- 总字数控制，不要超太多

### 3.4 贯彻执行题（公文写作）

**题型识别：**
- "写一份..."
- "拟一份..."
- "撰写..."
- 文种：讲话稿、宣传稿、倡议书、调研报告、工作方案等

**常见文种及格式：**

**1. 讲话稿/发言稿**
```
标题：关于XX的讲话/在XX会议上的讲话

称谓：各位领导、各位同事/同志们：

开头：开场白，说明背景、目的

主体：阐述观点、分析问题、提出要求

结尾：总结升华，表达期望

落款：讲话人、日期（可不写）
```

**2. 宣传稿**
```
标题：关于XX的倡议书/致XX的一封信

称谓：广大市民朋友们/全体市民：

开头：介绍背景，说明目的

主体：宣传内容（意义、做法、倡议）

结尾：呼吁号召

落款：发文单位、日期
```

**3. 调研报告**
```
标题：关于XX的调研报告

开头：调研背景、目的、时间、地点、方式

主体：
一、基本情况
二、主要做法
三、存在问题
四、对策建议

结尾：总结

落款：调研单位、日期
```

**4. 工作方案**
```
标题：XX工作实施方案/关于XX的工作方案

开头：背景、目的、依据

主体：
一、工作目标
二、工作原则
三、工作内容（分步骤、分阶段）
四、保障措施（组织、资金、人员）

落款：单位、日期
```

**5. 短评/时评**
```
标题：自拟标题（要有观点）

正文：
开头：引出话题，表明观点

主体：分析论证（原因、影响、例子）

结尾：总结升华

（不需要落款）
```

**答题步骤：**
1. 审题：明确文种、身份、对象、目的
2. 定格式：根据文种确定格式要素
3. 写内容：
   - 开头：背景、目的、依据
   - 主体：根据文种特点写（总结/分析/倡议等）
   - 结尾：总结、号召、展望
4. 检查：格式、语言、字数

**注意事项：**
- 格式正确（标题、称谓、落款）
- 身份恰当（语气符合身份定位）
- 对象明确（根据受众调整语言）
- 目的清晰（围绕目的组织内容）
- 控制字数，不要写太多

### 3.5 大作文

**考试要求：**
- 字数：1000-1500字（根据题目要求）
- 分值：40分（通常）
- 时间：建议60-70分钟

**文章结构：**

**五段三分式（推荐）：**
```
标题：简洁有力，体现中心

开头（150字左右）：
- 引出话题
- 提出中心论点

分论点一（200字左右）：
- 段首句（分论点）
- 阐释论证
- 举例/引用
- 小结

分论点二（200字左右）：
- 同上结构

分论点三（200字左右）：
- 同上结构

结尾（150字左右）：
- 总结全文
- 升华主题
- 展望未来/发出号召
```

**写作步骤：**

**第一步：审题（5分钟）**
- 看清题目要求：自拟标题/给定标题
- 确定主题：围绕什么写
- 明确立意：你的观点是什么

**第二步：列提纲（5分钟）**
- 确定标题
- 写出中心论点
- 确定三个分论点
- 想好论据素材

**第三步：写作（50-60分钟）**
- 开头：简洁明了，引出论点
- 分论点：每段一个中心句
- 论证：讲道理+举例子
- 结尾：呼应开头，升华主题

**第四步：检查（5分钟）**
- 字数是否够
- 有无错别字
- 语句是否通顺

**分论点设置角度：**

| 角度 | 示例 |
|------|------|
| 是什么-为什么-怎么做 | 科技创新的意义、原因、对策 |
| 个人-社会-国家 | 个人修养、社会氛围、国家制度 |
| 思想-制度-行动 | 转变观念、完善制度、落实行动 |
| 政府-企业-个人 | 政府监管、企业自律、个人参与 |
| 过去-现在-未来 | 历史回顾、当前现状、未来展望 |

**开头写法：**

**1. 引用式：**
> 习近平总书记指出："创新是引领发展的第一动力。"在新时代背景下，...（引出主题）

**2. 举例式：**
> 从高铁飞驰到5G商用，从移动支付到共享经济，...（举例引出主题）

**3. 对比式：**
> 过去...，现在...。这种变化启示我们...（引出主题）

**4. 背景式：**
> 当前，我国正处于...的关键时期。如何...，成为摆在我们面前的重要课题。（引出主题）

**5. 设问式：**
> 什么是...？如何在...？这是我们需要深入思考的问题。（引出主题）

**论证方法：**

**1. 举例论证：**
- 用具体事例证明观点
- 例子要典型、新颖
- 简单叙述，不要讲故事

**2. 道理论证：**
- 用理论、原理、政策说明
- 引用领导人讲话、政策文件
- 进行逻辑分析

**3. 对比论证：**
- 正反对比
- 古今对比
- 中外对比

**4. 引用论证：**
- 引用名言警句
- 引用诗词典故
- 引用数据报告

**结尾写法：**

**1. 总结式：**
> 综上所述，...只有...，才能...。让我们...，共同...！

**2. 展望式：**
> 展望未来，...。相信通过...，我们一定能够实现...！

**3. 号召式：**
> ...任重道远。让我们携起手来，以...，共同...！

**4. 升华式：**
> 从...到...，...不仅...，更是...。让我们...，为...贡献力量！

**高分技巧：**

1. **标题要亮**：
   - 对仗式：如"守正创新 砥砺前行"
   - 比喻式：如"以创新之光照亮前行之路"
   - 观点式：如"科技创新需要制度护航"

2. **论点要明**：
   - 分论点放在段首
   - 句式整齐，字数相近

3. **论据要新**：
   - 用最新的例子
   - 避免老掉牙的例子

4. **语言要准**：
   - 用政府语言
   - 避免口语化
   - 适当用排比、对偶

5. **卷面要整**：
   - 字迹工整
   - 段落分明
   - 标点正确

**注意事项：**
- 不要偏题，一定要紧扣材料
- 不要跑题，中心论点要明确
- 不要超字数，也不要太少
- 不要涂改太多，想好再写
- 不要照搬材料原文

## 四、申论学习建议

### 4.1 日常积累

**读什么：**
- 人民日报评论版
- 半月谈杂志
- 新华社评论员文章
- 求是杂志
- 学习强国APP

**怎么读：**
- 看结构：开头怎么引，中间怎么论，结尾怎么收
- 学语言：规范表达、高频词汇
- 记素材：好词好句、典型案例

### 4.2 规范表达积累

**高频词汇：**

**思想层面：**
提高认识、统一思想、转变观念、树立意识、高度重视

**制度层面：**
完善制度、健全机制、建立体系、规范管理、明确责任

**执行层面：**
加强领导、落实责任、强化监督、严格考核、持续改进

**保障层面：**
加大投入、提供支持、创造条件、营造氛围、形成合力

**效果层面：**
取得实效、显著改善、大幅提升、明显增强、持续推进

### 4.3 练习方法

**小题练习：**
- 每天1-2道小题（概括、分析、对策）
- 严格限时（每题20-30分钟）
- 做完对答案，找出差距
- 反复修改，直到满意

**大作文练习：**
- 每周1篇完整作文
- 严格限时（70分钟）
- 手写，不要打字
- 找人批改或自己对照范文修改

**真题使用：**
- 近5年国考真题必做
- 目标省份近5年真题必做
- 其他省份真题选做
- 至少做2-3遍

### 4.4 答题时间分配

| 题型 | 建议时间 |
|------|---------|
| 阅读材料 | 30-40分钟 |
| 概括题（每道） | 20-25分钟 |
| 分析题 | 25-30分钟 |
| 对策/公文题 | 30-35分钟 |
| 大作文 | 60-70分钟 |
| 检查 | 10分钟 |
| **总计** | **180分钟** |

### 4.5 答题卡书写技巧

**第一题（概括题）：**
- 200-300字
- 写10-15行
- 分条作答，每条一行

**第二题（分析题）：**
- 200-300字
- 写10-15行
- 分段作答

**第三题（对策/公文题）：**
- 400-500字
- 写20-25行
- 公文注意格式

**第四题（大作文）：**
- 1000-1200字
- 写40-50行
- 5-7段为宜

**注意事项：**
- 字迹工整，大小适中
- 段落分明，不要太挤
- 标清题号，不要写错位置
- 控制字数，超出格子不算分

## 五、申论误区与纠正

### 误区1：申论就是抄材料

**问题：** 大量抄材料原文
**纠正：** 
- 可以借鉴材料观点，但要自己组织语言
- 提炼概括，不要整段抄
- 大作文要结合材料但要有自己的观点

### 误区2：申论靠模板套话

**问题：** 背一堆万能模板，生搬硬套
**纠正：**
- 模板可以参考，不能依赖
- 要具体问题具体分析
- 答题要结合材料，有针对性

### 误区3：只看不写

**问题：** 看很多范文，自己不动笔
**纠正：**
- 申论必须动手写
- 光看是提高不了的
- 每周至少写1篇大作文

### 误区4：大作文越华丽越好

**问题：** 堆砌辞藻，空洞无物
**纠正：**
- 内容重于形式
- 观点明确、论证充分更重要
- 语言通顺即可，不要过度追求华丽

### 误区5：小题写的越多越好

**问题：** 超字数，写很多不相关的内容
**纠正：**
- 严格按字数要求
- 精炼表达，去掉废话
- 要点在精不在多

### 误区6：忽视字迹和卷面

**问题：** 字迹潦草，涂改严重
**纠正：**
- 字迹工整很重要
- 想好了再写，减少涂改
- 卷面整洁能加分
