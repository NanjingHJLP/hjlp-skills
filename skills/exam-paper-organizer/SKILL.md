# 模拟题组织

为用户提供按知识点/难度组卷、生成答案解析和成绩分析的完整方案，覆盖小学、中学、大学各阶段主要学科。

## 使用场景

- 组织特定章节/范围的模拟试题（如"大学一年级高数前5章期中模拟"）
- 按知识点分布组卷（如"覆盖三角函数、解析几何、立体几何"）
- 按难度梯度组卷（基础题60%、中等题30%、难题10%）
- 生成配套答案与详细解析
- 提供成绩分析与学习建议

## 组卷流程

### Step 1: 明确需求参数

收到组卷请求时，确认以下信息：

| 参数 | 说明 | 示例 |
|-----|------|------|
| 学段 | 小学/初中/高中/大学 | 大学一年级 |
| 学科 | 具体科目 | 高等数学 |
| 考试类型 | 阶段测验/期中/期末/模拟 | 期中模拟 |
| 覆盖范围 | 章节/知识点 | 第1-5章 |
| 总分/时长 | 试卷规格 | 100分/120分钟 |
| 难度分布 | 基础:中等:难题比例 | 5:3:2 |
| 题型要求 | 选择题/填空/解答等 | 选择题30分+解答题70分 |

**若用户未明确，主动询问补充。**

### Step 2: 设计试卷结构

根据考试类型设计标准结构：

**大学公共课标准结构（100分/120分钟）**
- 选择题：10题×3分=30分（基础概念）
- 填空题：6题×4分=24分（计算/推导）
- 解答题：4-5题（46分，含证明/应用/综合）

**高中标准结构（150分/120分钟）**
- 单选题：8题×5分=40分
- 多选题：4题×5分=20分（选不全得2分）
- 填空题：4题×5分=20分
- 解答题：6题（70分，含选做题）

**初中标准结构（120分/120分钟）**
- 选择题：10题×3分=30分
- 填空题：6题×3分=18分
- 解答题：8-9题（72分）

### Step 3: 知识点权重分配

根据覆盖范围，合理分配各章节分值：

**原则：**
- 重点章节分值较高
- 与后续学习关联度高的章节增加权重
- 避免某章节过度集中或缺失

**示例分配：**
大学高数第1-5章期中（极限、导数、微分中值定理、不定积分、定积分）：
- 第1章（极限）：15% → 约15分
- 第2章（导数）：20% → 约20分
- 第3章（微分中值定理）：20% → 约20分
- 第4章（不定积分）：20% → 约20分
- 第5章（定积分）：25% → 约25分

### Step 4: 生成试题

按以下标准生成每道题目：

**选题原则：**
1. **代表性**：选择该知识点典型题型
2. **梯度性**：同一知识点内设置不同难度
3. **综合性**：解答题体现多知识点融合
4. **新颖性**：避免陈题，注重情境创新

**难度控制：**
- 基础题：直接套用公式/定理，计算量小
- 中等题：需要1-2步转化，或中等计算量
- 难题：需要多步推导、技巧性强、或综合多个知识点

### Step 5: 生成答案解析

每题提供完整解析：

**答案格式：**
1. **答案**：明确给出最终答案
2. **考点**：指出考查知识点
3. **解析**：详细推导过程
4. **易错点**（选填/中等难度以上）：常见错误提醒
5. **技巧点拨**（难题）：解题技巧总结

### Step 6: 成绩分析模板

提供成绩分析与学习建议：

**分数段诊断：**
| 得分率 | 诊断 |
|-------|------|
| 90%-100% | 优秀，知识体系完整，可挑战更高难度 |
| 80%-89% | 良好，基础扎实，个别细节需加强 |
| 70%-79% | 中等，存在知识漏洞，需针对性复习 |
| 60%-69% | 及格，基础薄弱，建议系统梳理 |
| <60% | 不及格，需重新学习基础概念 |

**错题分析维度：**
- 知识点分布（哪个章节错题最多）
- 题型分析（选择题/解答题失分情况）
- 难度分析（基础/中等/难题掌握情况）
- 错误类型（概念不清/计算错误/思路偏差/时间不足）

## 组卷示例

### 示例1：大学高数期中模拟

**用户需求：** "大学一年级上学期高数期中模拟题，覆盖前5章内容"

**输出结构：**
1. 试卷（100分，120分钟）
   - 选择题10题×3分（极限计算、导数定义、积分性质）
   - 填空题6题×4分（求极限、求导数、求积分）
   - 解答题4题（证明题、综合计算、应用题）
2. 答案与解析（每题详细解答）
3. 成绩分析报告模板

### 示例2：高中数学期末模拟

**用户需求：** "高二数学期末模拟，重点考查圆锥曲线和导数"

**输出结构：**
1. 试卷（150分，120分钟）
   - 按高考标准题型配置
   - 圆锥曲线40分、导数35分、其他章节75分
2. 答案与解析
3. 成绩分析

## 注意事项

1. **题目原创性**：模拟题应基于经典题型改编，避免直接复制真题
2. **难度适中**：模拟题难度应与实际考试相当或略高
3. **答案准确**：确保答案无误，解析步骤完整
4. **格式规范**：试卷排版清晰，题号、分值标注明确
5. **循序渐进**：首次组卷可先做基础题，逐步增加难度
