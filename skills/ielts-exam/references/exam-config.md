# 雅思考试配置

## 考试类型

### 学术类（Academic）
- **适用人群：** 大学申请者、职业注册（医生、护士等）
- **写作 Task 1：** 描述图表、表格、流程图（150 词以上）
- **阅读文章：** 学术期刊、书籍、报纸中的学术文本

### 培训类（General Training）
- **适用人群：** 移民（加拿大、澳大利亚、新西兰）、工作体验项目
- **写作 Task 1：** 写书信（正式/半正式/非正式，150 词以上）
- **阅读文章：** 日常生活文本 — 广告、手册、通知、职场文件

## 模块结构

| 模块 | 时长 | 题数 | 形式 |
|------|------|------|------|
| 听力 | 30 分钟（+10 分钟誊写） | 40 题 | 4 段录音，多种题型 |
| 阅读 | 60 分钟 | 40 题 | 3 篇文章，多种题型 |
| 写作 | 60 分钟 | 2 题 | Task 1（20 分钟），Task 2（40 分钟） |
| 口语 | 11-14 分钟 | 3 部分 | 面对面面试 |

## 听力详情

**4 个 Section：**
1. 日常社交对话（2 人）
2. 日常社交独白（1 人）
3. 教育/培训场景（最多 4 人）
4. 学术讲座/独白（1 人）

**题型：**
- 选择题
- 配对题
- 地图/图表标注
- 表格/笔记/填空
- 句子填空
- 简答

## 阅读详情

**学术类：** 3 篇长文章（共 2,150-2,750 词）
**培训类：** 3 个部分（短文章 → 中等文章 → 长文章）

**题型：**
- True/False/Not Given
- Yes/No/Not Given
- 段落标题配对
- 信息配对
- 特征配对
- 句子填空
- 摘要填空
- 选择题
- 简答

## 写作详情

### Task 1（20 分钟，150 词以上）

| 学术类 | 培训类 |
|--------|--------|
| 图表、表格、流程图 | 正式书信（投诉、请求） |
| 比较、对比、总结趋势 | 半正式书信（给熟人） |
| 报告风格，客观 | 非正式书信（给朋友/家人） |

### Task 2（40 分钟，250 词以上）

学术类和培训类相同：就观点、论点或问题写论述文
- 同意/不同意
- 讨论双方观点
- 优缺点
- 问题/解决方案
- 双问题

## 口语详情

| 部分 | 时长 | 形式 |
|------|------|------|
| Part 1 | 4-5 分钟 | 自我介绍 + 熟悉话题（家乡、工作、学习） |
| Part 2 | 3-4 分钟 | 长篇独白：1 分钟准备，2 分钟按提示卡讲述 |
| Part 3 | 4-5 分钟 | 讨论：与 Part 2 相关的抽象问题 |

## 分数等级

| 分数 | 水平 | 描述 |
|------|------|------|
| 9 | 专家级 | 完全掌握 |
| 8 | 优秀 | 偶有不准确 |
| 7 | 良好 | 能应对复杂情况，偶有错误 |
| 6 | 合格 | 总体有效，有错误 |
| 5 | 中等 | 部分掌握，错误较多 |
| 4 | 有限 | 熟悉场景下的基本能力 |
| 3 | 极有限 | 仅能传达大概意思 |
| 2 | 间歇 | 理解困难 |
| 1 | 非使用者 | 仅能使用孤立单词 |

## 分数计算

**听力 & 阅读：** 原始分（满分 40）换算为等级分

| 原始分 | 等级分 |
|--------|--------|
| 39-40 | 9.0 |
| 37-38 | 8.5 |
| 35-36 | 8.0 |
| 33-34 | 7.5 |
| 30-32 | 7.0 |
| 27-29 | 6.5 |
| 23-26 | 6.0 |

**写作 & 口语：** 考官按 4 项标准评分，取平均后四舍五入到最近的 0.5

**总分：** 4 项平均，四舍五入到最近的 0.5

## 考试信息

| 项目 | 详情 |
|------|------|
| 报名 | 通过 British Council、IDP 或雅思官网在线报名 |
| 形式 | 机考（3-5 天出分）或纸笔（13 天出分） |
| 费用 | 约 245-310 美元（各国不同） |
| 成绩有效期 | 2 年 |
| 重考 | 无等待期（可报最近场次） |

## 单科重考（OSR）

- 原考后 60 天内可申请
- 仅重考一个模块（听力、阅读、写作或口语）
- 新总分按新模块分数计算
- 须与原考试类型相同（学术类或培训类）
- 费用低于全科重考
- **适用场景：** 某一模块拉低总分时使用
