# 卦象解读方法指南

本文件提供系统的卦象解读方法，包括变爻规则、解读维度和实操框架。

---

## 解读八维度

每个卦象可从以下八个维度进行解读：

| 维度 | 内容 | 何时使用 |
|------|------|----------|
| 卦辞 | 整体卦象的总断 | 每次必用 |
| 彖辞 | 对卦辞的义理解释 | 需要深入理解时 |
| 象辞 | 象征意义与君子之道 | 需要行为指导时 |
| 爻辞 | 每一爻的具体解释 | 有变爻时重点使用 |
| 变爻 | 动爻的变化分析 | 有变爻时必用 |
| 互卦 | 内在变化趋势 | 需要深层分析时 |
| 错卦 | 阴阳全变，对立视角 | 需要反面思考时 |
| 综卦 | 上下颠倒，反向视角 | 需要换位思考时 |

---

## 变爻分析规则

变爻数量不同，解读方法不同。这是传统的核心规则：

### 无变爻

```text
规则：以本卦卦辞为主进行解读
说明：卦象稳定，关注当前状态
```

### 一个变爻

```text
规则：以变爻的爻辞为主要依据
说明：这是最清晰的指引，变爻直接点明关键

示例：得乾卦，九三变
→ 重点看九三爻辞：「君子终日乾乾，夕惕若厉，无咎」
→ 解读：需要持续努力并保持警惕
```

### 两个变爻

```text
规则：以上面的变爻爻辞为主，下面的变爻为辅
说明：两个变爻形成主次关系

示例：得乾卦，九三和九五变
→ 主看九五：「飞龙在天，利见大人」
→ 辅看九三：「夕惕若厉」
→ 解读：有成功机会，但过程中需保持警惕
```

### 三个变爻

```text
规则：以本卦卦辞为主，变卦卦辞为参考
说明：变化较多，回归整体判断

示例：得泰卦，三个爻变为否卦
→ 主看泰卦卦辞：「小往大来，吉亨」
→ 参考否卦卦辞：「不利君子贞」
→ 解读：当前通达，但有向闭塞转变的趋势
```

### 四个变爻

```text
规则：以变卦中两个不变的爻为主，下爻优先
说明：不变的爻反而是关键

示例：得乾卦，初九和九二不变（其余四爻皆变）
→ 主看变卦中对应位置的爻辞
→ 下面的不变爻优先
```

### 五个变爻

```text
规则：以变卦中唯一不变的爻为主
说明：唯一不变的爻是核心指引
```

### 六个变爻（全变）

```text
规则：以变卦卦辞为主
特殊：乾卦六爻全变用「用九」，坤卦六爻全变用「用六」

乾卦用九：「见群龙无首，吉」
坤卦用六：「利永贞」
```

### 速查表

| 变爻数 | 主要依据 | 辅助参考 |
|--------|----------|----------|
| 0 | 本卦卦辞 | — |
| 1 | 变爻爻辞 | 本卦卦辞 |
| 2 | 上变爻爻辞 | 下变爻爻辞 |
| 3 | 本卦卦辞 | 变卦卦辞 |
| 4 | 变卦不变爻（下优先） | 变卦卦辞 |
| 5 | 变卦唯一不变爻 | 变卦卦辞 |
| 6 | 变卦卦辞（乾坤用九用六） | — |

## 互卦推导

互卦揭示事物内在的变化趋势。

### 推导方法

```text
本卦六爻：初 二 三 四 五 上

互卦下卦 = 本卦 二、三、四 爻
互卦上卦 = 本卦 三、四、五 爻
```

### 示例

```text
泰卦（☷☰）：
六爻从下到上：阳 阳 阳 阴 阴 阴

互卦下卦 = 二三四 = 阳阳阴 = 兑 ☱
互卦上卦 = 三四五 = 阳阴阴 = 震... 

（实际推导需根据具体爻的阴阳）
```

### 解读意义

互卦代表事物发展过程中内在的、不易察觉的趋势。当本卦和变卦的信息不够明确时，互卦可以提供补充视角。

## 错卦与综卦

### 错卦（旁通卦）

```text
方法：将本卦每一爻阴阳互换
意义：从对立面看问题

示例：
乾（☰☰）的错卦 = 坤（☷☷）
泰（☷☰）的错卦 = 否（☰☷）
```

### 综卦（覆卦）

```text
方法：将本卦上下颠倒
意义：换一个角度看同一件事

示例：
屯（☵☳）的综卦 = 蒙（☶☵）
→ 屯是创业的艰难，蒙是成长的迷茫——同一阶段的两个视角
```

### 应用场景

- 解读遇到困惑时，看错卦获得对立视角
- 想了解对方立场时，看综卦换位思考
- 深度分析时，本卦 + 变卦 + 互卦 + 错卦 + 综卦 = 全方位理解

## 解读实操框架

### 标准解读流程

```text
Step 1：确认卦象
  → 本卦是什么？变爻在哪？变卦是什么？

Step 2：读卦辞
  → 本卦卦辞说了什么？整体吉凶如何？

Step 3：看象辞
  → 象辞给出什么行为指导？

Step 4：析变爻
  → 按变爻规则确定主要依据
  → 变爻爻辞的具体含义

Step 5：观变卦
  → 变卦代表什么趋势？
  → 从本卦到变卦，事态如何发展？

Step 6：综合判断
  → 结合用户问题，给出针对性分析
  → 明确行动建议
```

### 解读注意事项

| 要做 | 不要做 |
|------|--------|
| 引用原文再解释 | 凭空编造卦辞 |
| 结合具体问题分析 | 泛泛而谈的套话 |
| 说明吉凶的依据 | 武断的好坏判断 |
| 承认解读的多义性 | 声称只有一种解释 |
| 给出可行的建议 | 模糊的「顺其自然」 |
| 提醒是参考非预测 | 暗示命中注定 |

## 特殊卦象处理

### 八纯卦（上下卦相同）

```text
乾为天、坤为地、震为雷、巽为风、
坎为水、离为火、艮为山、兑为泽

特点：力量纯粹但容易走极端
解读：注意「亢」的风险，物极必反
```

### 泰否二卦

```text
泰（☷☰）：天地交 → 通达
否（☰☷）：天地不交 → 闭塞

这两卦常用于判断大势，是否泰来否极
```

### 既济未济

```text
既济（☵☲）：已完成 → 注意守成
未济（☲☵）：未完成 → 继续努力

这两卦常用于判断事情的完成度
```
