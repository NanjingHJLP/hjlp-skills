# 学习方法与练习

## 先诊断

制定学习计划前：

1. **进行诊断** — 完整模拟考或模块专项评估
2. **确定各模块分数** — 使用官方分数换算表
3. **计算差距** — 当前与目标分数对比
4. **按投入产出比排序** — 差距大且提升潜力大的模块优先

## 分模块练习

### 听力

**常见问题：**
- 阅读下题时错过答案
- 填空题拼写错误
- 长录音中迷失位置

**练习方法：**
- 速度训练：1.25 倍速听，考试时感觉更慢
- 预测训练：先读题，预测答案类型
- 拼写清单：整理学术词汇拼写
- Section 4 专项：学术讲座最难，多练

**题型专项训练：**
先针对最弱题型单独训练，再进行完整测试。

### 阅读

**常见问题：**
- 时间不够（最常见）
- True/False/Not Given 混淆
- 段落标题配对耗时过长

**练习方法：**
- 限时训练：每篇 20 分钟，严格截止
- 略读训练：2 分钟内找出主旨
- T/F/NG 逻辑："Not Given" = 信息**缺失**，而非未提及
- 关键词定位：划线标记，不要重读

**时间管理：**
- 文章 1：15 分钟（最简单）
- 文章 2：20 分钟
- 文章 3：20 分钟（最难）
- 缓冲：5 分钟誊写/检查

### 写作

**Task 1（学术类）检查清单：**
- [ ] 首段有概述（主要趋势/特征）
- [ ] 选取关键特征（非全部数据）
- [ ] 进行比较（更高、更低、相似）
- [ ] 数据引用准确
- [ ] 150 词以上，最多 20 分钟

**Task 1（培训类）检查清单：**
- [ ] 语气正确（正式/半正式/非正式）
- [ ] 回答所有要点
- [ ] 开头表明目的
- [ ] 开头和结尾得体
- [ ] 150 词以上，最多 20 分钟

**Task 2 检查清单：**
- [ ] 首段清晰表明立场
- [ ] 每段一个中心思想
- [ ] 论点有例证支撑
- [ ] 结论重申立场并总结
- [ ] 250 词以上，最多 40 分钟

**写作 7 分修正：**

| 问题 | 修正 |
|------|------|
| "Firstly, Secondly, Thirdly" | 变化："A key consideration...", "Equally important...", "Furthermore..." |
| 段落开头用 "However" | 用主题句开头，连接词放第二 |
| 笼统例子 | 具体："In Finland, students..." 而非 "In some countries..." |
| 结论太短 | 至少 2 句：重申立场 + 收尾思考 |

### 口语

**Part 1（4-5 分钟）：**
- 扩展回答：每题 2-3 句
- 不要背稿（考官能察觉）
- 练习常见话题：家乡、工作、学习、爱好

**Part 2（3-4 分钟）：**
- 利用 1 分钟准备：记下所有提示要点
- 结构：开场 → 逐一回应提示 → 收尾
- 说满 2 分钟（沉默会被扣分）
- 录下自己，回听改进

**Part 3（4-5 分钟）：**
- 预期 "why" 和 "how" 问题
- 表达观点后用理由支撑
- 推测："It's likely that...", "This could be because..."
- 需要时可要求澄清（显示自信）

**流利度杀手（需消除）：**
- "Um", "uh", "like"（填充词）
- 长停顿（>2 秒）
- 重复句首（"I think... I think... I think..."）
- 过度自我纠正（"No, wait, I mean..."）

## 学习计划框架

**考试前 8 周（差距 >2 分）：**

| 周次 | 重点 |
|------|------|
| 1-2 | 诊断 + 最弱模块强化 |
| 3-4 | 第二弱模块 + 保持第一模块 |
| 5-6 | 完整模拟考 + 专项训练 |
| 7 | 全模块，轻度练习 |
| 8 | 轻度复习，考前休息 |

**考试前 4 周（差距 <1 分）：**

| 周次 | 重点 |
|------|------|
| 1-2 | 针对薄弱点专项训练 |
| 3 | 完整模拟考 |
| 4 | 轻度复习，考前休息 |

**职场人士每日最低量：**
- 15 分钟：词汇复习
- 15 分钟：一项专项训练（如 5 道 T/F/NG）
- 周末：1 次完整模拟考
