# 漫画图像生成 Prompt 指南

## 核心原则

漫画图像生成 prompt 应包含以下要素：
1. **场景描述**：用户提供的原始文本
2. **风格指定**：素描（sketch）或彩绘（color）
3. **质量修饰词**：确保图像清晰、构图合理
4. **漫画特征**：强调漫画风格、画面感

## 素描风格 Prompt 模板

```
[场景描述]，漫画风格，素描，黑白线稿，清晰线条，背景丰富，人物生动，高质量，细节精细
```

**关键词**：
- 素描：sketch, black and white, line drawing
- 线条：clean lines, detailed linework
- 风格：comic style, manga style
- 质量：high quality, detailed, professional

**示例**：
- 原始文本："一个英雄站在山顶上，眺望远方"
- Prompt："一个英雄站在山顶上，眺望远方，漫画风格，素描，黑白线稿，清晰线条，背景丰富，人物生动，高质量"

## 彩绘风格 Prompt 模板

```
[场景描述]，漫画风格，彩绘，色彩鲜艳，细节丰富，漫画书风格，动态构图，高质量
```

**关键词**：
- 彩绘：color, vibrant colors, detailed coloring
- 风格：comic book style, manga style
- 画面：dynamic composition, detailed background
- 质量：high quality, professional illustration

**示例**：
- 原始文本："英雄拔出剑，准备战斗"
- Prompt："英雄拔出剑，准备战斗，漫画风格，彩绘，色彩鲜艳，细节丰富，漫画书风格，动态构图，高质量"

## 场景类型优化建议

### 动作场景
添加："动态动作，激烈战斗，运动线条，速度感"

### 情感场景
添加："情感表达，面部表情细腻，氛围感"

### 风景场景
添加："广阔背景，透视感，环境细节"

## 避免的问题

1. **文字重叠**：避免在图像中生成文字（漫画对话框除外）
2. **画面混乱**：提示词不要太复杂，保持焦点
3. **风格不一致**：确保所有页面使用相似风格

## 调用 image_generate 工具的参数建议

- **ratio**：推荐 16:9 或 4:3（横屏漫画）
- **size**：默认 2K 即可
- **prompt**：按上述模板构建

## 特殊技巧

1. **多角色场景**：明确描述角色关系和位置
2. **特定时代背景**：添加如"中世纪城堡"、"未来都市"等
3. **光影效果**：可指定"戏剧性光影"、"柔和光线"等
