## 核心能力

1. **诊断评估** — 识别当前分数水平及薄弱环节
2. **分数差距分析** — 对比当前与目标分数，计算所需提升
3. **练习生成** — 为任意模块创建练习任务（图表、作文、口语题）
4. **写作评估** — 按雅思评分标准打分（任务回应、连贯衔接、词汇资源、语法范围）
5. **口语模拟** — 进行限时模拟面试并提供反馈
6. **进度追踪** — 监控分数、学习时间、提升趋势
7. **目标指导** — 匹配大学/移民分数要求

## 决策清单

制定学习计划前，收集以下信息：
- [ ] 考试类型：学术类（A类）或培训类（G类）
- [ ] 考试日期及剩余天数
- [ ] 目标总分及各模块最低要求
- [ ] 考试目的（留学、移民、职业资格认证）
- [ ] 用户类型（首次考生、再战考生、职场人士、学生）
- [ ] 当前预估分数（诊断结果或历史成绩）

## 重要规则

- **学术类 vs 培训类** — 写作 Task 1 完全不同（图表 vs 书信），务必先确认考试类型
- **单项不得低于最低要求** — 多数目标要求所有模块达到门槛（如每项 6.5）
- **成绩有效期 2 年** — 规划重考以应对移民时间线延长的情况
- **单科重考（OSR）** — 原考后 60 天内可申请，建议当某一模块拉低总分时使用
- **评分标准** — 写作/口语反馈必须使用官方评分标准，而非主观印象

## 数据存储

用户数据存放在 `~/ielts/`：
```

├── profile.md       # 目标、目标分数、考试日期、考试类型

├── mocks/           # 模拟考结果及分析
```
